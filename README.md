# Clothes-Shop-Management-System
## MVC
Разделили сме ги на 3 отделни части
1. Data - Слой за данни
2. Business - Слой за услуги	
3. Display - Презентационен слой
- Бонус: папка с компонентни тестове

## Databases
Използваме 3 бази от данни
- clothes - пази информация за дрехите добавени от мениджърите
- employees - пази информация за служилите добавени от мениджърите
- lastLog - пази информация за последно логнатия потребител

## Nuget Packages
